# 👑 King's Gold

A narrative-driven puzzle adventure. The royal treasury of **Liitokala** has been
stolen and scattered across **30 countries**. As the King's *World Tracker*, you
recover the gold by choosing the right animal companion for each trial — while
keeping your **Heat** (suspicion) low enough to avoid the thieves and their allies.

Built as a lightweight, **mobile-first HTML5** game with **no backend required**
(Firebase cloud-save is optional). Plays entirely from static files.

## Play

It's a static site — serve the folder over HTTP and open `index.html`:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

(Opening via `file://` won't work because the game `fetch`es its JSON data.)

## Game loop

```
Title → Intro (meet the King, name your hero)
      → Country Map (Chapter 1 of 30)
        → 5 puzzles per country (Arrival Hazard, Ally Test, Cryptic Artifact,
          Chase/Stealth, Lockbox)
          → pick a companion → GOOD / NORMAL / BAD outcome
          → a bad streak summons an enemy encounter
      → Country Summary → next country
→ Final Ending (Golden Triumph / Indentured Middle / The Fall Guy)
```

**Outcomes** — GOOD `+3% Gold` (resets bad streak) · NORMAL `+1% Gold` ·
BAD `+4 Heat` (and grows your consecutive-bad streak).

**Enemies** — 2 consecutive bad → Street Brigand · 3 → Royal Turncoat ·
4+ → False Vizier. Win to cool your Heat; lose and it spikes.

**Endings** — `Golden Triumph` (≥80% gold & ≤10 heat) ·
`The Fall Guy` (<30% gold **or** ≥26 heat) · `The Indentured Middle` (otherwise).

## How content is built

150 encounters (30 countries × 5 slots) are generated by a small, data-driven
engine instead of 150 hand-written files:

- **`scripts/data/pets.json`** — 24 companions (Land/Sea/Sky), each with ability
  **tags** (`fire`, `water`, `speed`, `stealth`, …).
- **`scripts/data/countries.json`** — 30 chapters (name, landmark, signature biome).
- **`scripts/data/baddies.json`** — the three enemy encounters.
- **`scripts/engine.js`** — combines a country + slot into a themed hazard, then
  sorts the 24 pets into GOOD / NORMAL / BAD by matching ability tags to the
  hazard (e.g. *water/ice* beats a wildfire; *fire* makes it worse).

## Architecture

- **`scripts/router.js`** — hash-based scene router (`navigate()` force-renders
  on same-hash transitions, e.g. puzzle slot → next slot).
- **`scripts/state.js`** — runtime state + persistence. **localStorage is the
  source of truth**; Firebase is mirrored best-effort and never blocks play.
- **`scripts/hud.js`** — persistent Gold % / Heat top bar.
- **`scripts/scenes/`** — `title`, `intro`, `map`, `puzzle`, `encounter`,
  `summary`, `ending`, `gameover`.
- **`scripts/main.js` / `save.js`** — optional Firebase, loaded **lazily** so an
  unreachable CDN can never stop the game from loading.

## Saving

Saves are keyed by the explorer name you enter. **Continue** from the title
screen reloads your gold, heat, country, and slot from localStorage.
